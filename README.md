## ✨ 它是什么？

这是一个**可直接安装在 [扣子(Coze)](https://www.coze.cn) 平台上的AI技能(Skill)**，也可以作为独立的Prompt方法论用于任何AI图像/视频生成工具。

输入一句"我想要一个XX场景的压迫感镜头"，它会按专业电影摄影逻辑输出**中英文双版本**完整Prompt，开箱即用。

## 🎯 核心方法论：五层拆解法 + 两大流派

传统AI绘画教程只给你"低角度+广角"的套话，但真正的压迫感来自**精确的镜头语言组合 + 视觉心理学**。本技能识别出两种根本不同的压迫感范式：

| 流派 | 核心组合 | 心理学机制 | 代表场景 |
|------|---------|-----------|---------|
| **A. 巨物扑来派（超广角）** | 低角度仰拍 + 14-24mm超广角 + 大远景 + 大景深 | 夸张近大远小透视，巨物从上方向观众"压"下来 | Boss登场、巨兽出山、海啸压城、飞船遮天 |
| **B. 幽闭窒息派（长焦压缩）** | 胸口以下低机位 + 15°仰角 + 200mm+超长焦 + 复合构图(威胁4/5占比) + 浅景深(焦点锁威胁核心) + 背景融化 | 空间压缩抹除安全距离，威胁"贴脸"无处可逃 | 直面怪兽巨口、黑洞在前、被军队包围 |

五层拆解维度：
1. 📷 **摄影四维度**：角度 + 焦距 + 景别 + 景深 (+机位高度)
2. 👹 **主体双公式**：压迫源 vs 被压迫者（量化画面占比）
3. 🌪️ **环境三层**：前景粒子遮挡 → 中景核心对峙 → 后景纵深/融化
4. 💡 **光影配方**：光源方向 + 光质 + 色温 + 氛围光 + 阴影
5. 🧠 **心理学校验**：7个自检问题确保每个视觉决策都在服务情绪

## 📦 内置7套场景模板

- **模板A** · 巨物压迫（Boss出场/巨兽逼近/巨型建筑）— 超广角派
- **模板B** · 战场史诗（千军万马/战争场面）— 超广角派
- **模板C** · 悬疑不安（未知威胁/恐怖场景）— 荷兰角
- **模板D** · 自然灾害/末日（海啸/火山/风暴/小行星）— 超广角派
- **模板E** · 科幻赛博压迫（巨型飞船/未来城市/AI统治）— 鱼眼超广角派
- **模板F** · 神圣/宗教压迫（巨型神像/圣殿/神降临）— 超广角派
- **模板G** · 幽闭窒息（长焦压缩派·原教程沙虫范式）— 长焦派 ⭐

## 🚀 安装到Coze

### 方式1：Coze商店安装（推荐）
在扣子技能商店搜索「压迫感镜头Prompt生成器」直接安装。

### 方式2：本地上传
1. 下载本仓库所有文件
2. 在Coze工作台 → 技能 → 创建技能 → 选择"上传ZIP包"
3. 上传 `pressure-shot-prompt.skill` 压缩包
4. 开始使用

## 💡 使用示例

**输入**：
> "给我一个《摸金校尉》下地宫遇到巨型粽子的压迫感镜头"

**输出**（节选）：
- 🎯 画面描述
- 📷 镜头语言解析（焦段/角度/景深/构图选择理由）
- ✨ 中文版完整Prompt（可直接复制到即梦/可灵）
- 🌍 English Prompt（可直接复制到Midjourney/Sora）
- 🎬 视频运镜建议（推镜/手摇/缓移等）
- 🧠 设计说明（3-5条心理学依据）
